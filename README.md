# dhwiki
An AI-powered knowledge base built specifically for SMBs. It supports private on-premises deployment, encrypted file storage, and the creation of personal private repositories. With open APIs, it seamlessly integrates with existing business systems such as ERP, CRM, and OA. It is compatible with all major document formats and delivers an AI Q&A accuracy rate of over 95%.

专门针对中小企业而研发的AI知识库，私有化部署，文件加密存储，支持创建个人私密库，有开放API，方便ERP/CRM/OA等系统对接，支持市面上的所有文档格式。AI问答准确率95%以上。
